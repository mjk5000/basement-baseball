BASEMENT BASEBALL - SETUP INSTRUCTIONS
=======================================

ACCESSING THE GAME
------------------
The game is hosted on GitHub Pages and can be accessed from any device:

Website URL: https://mjk5000.github.io/basement-baseball/

This works on:
- iPad (Safari, Chrome)
- Kindle Fire (Silk Browser, Chrome)
- Desktop/Laptop computers
- Android tablets and phones
- iPhones

No installation required - just open the URL in your browser!


GITHUB PAGES SETUP (Already Complete)
--------------------------------------
The game is already deployed and live. If you need to update it in the future:

1. Make changes to the files locally
2. Commit and push changes:
   git add .
   git commit -m "Description of changes"
   git push origin main

3. GitHub Pages will automatically update within 1-2 minutes


OFFLINE USE (PWA - Progressive Web App)
----------------------------------------
You can install the game as an app on your device:

On iPad/iPhone:
1. Open https://mjk5000.github.io/basement-baseball/ in Safari
2. Tap the Share button
3. Tap "Add to Home Screen"
4. Name it "Basement Baseball"
5. Tap "Add"

On Android/Kindle Fire:
1. Open the URL in Chrome
2. Tap the menu (3 dots)
3. Tap "Add to Home screen" or "Install app"
4. Confirm installation

On Desktop (Windows/Mac/Linux):

Using Chrome:
1. Open https://mjk5000.github.io/basement-baseball/ in Chrome
2. Look for the install icon (⊕ or computer icon) in the address bar (right side)
   - OR click the three dots menu → "Install Basement Baseball..."
3. Click "Install" in the popup
4. The app will open in its own window
5. A desktop shortcut and Start Menu entry will be created

Using Microsoft Edge:
1. Open the URL in Edge
2. Click the install icon (+) in the address bar
   - OR click the three dots menu → "Apps" → "Install this site as an app"
3. Click "Install"
4. The app appears in your Start Menu and taskbar

Using Safari (Mac):
1. Open the URL in Safari
2. Click "File" → "Add to Dock"
3. The game will be added as a web app in your Dock

Once installed, the desktop app will:
- Open in its own window (no browser tabs/address bar)
- Have its own icon in Start Menu/Dock/Applications
- Work offline after first load
- Feel like a native desktop application
- Support keyboard shortcuts and notifications

To uninstall desktop app:
- Windows: Right-click the app → "Uninstall"
- Mac: Right-click in Dock → "Options" → "Remove from Dock"
- Or use browser settings → Apps → Remove


RECOMMENDED SETTINGS FOR TABLETS
---------------------------------
1. Use landscape orientation for best experience
2. Enable sound for the full experience (unmute button in game)
3. Adjust brightness to comfortable level for extended play
4. Consider enabling full-screen mode (icon button in-game)

TROUBLESHOOTING
---------------
If the game doesn't load:
1. Check your internet connection
2. Try refreshing the page (pull down on mobile)
3. Clear browser cache
4. Try a different browser

If installed as PWA and not updating:
1. Uninstall the app
2. Clear browser cache
3. Reinstall from the website
